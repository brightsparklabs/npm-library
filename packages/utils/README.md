# BSL Common Typescript Utils

BSL standard library containing commonly used Typescript utilities.

## Installation

Package management is handled by [Bun](https://bun.com), you can install the dependencies with:

```bash
bun install
```

## Testing

Testing is handled by [Vitest](https://vitest.dev), they can be ran using:

```bash
bun run test
```

## Build

Building is handled by [Vite](https://vite.dev/), to build the project use:

```bash
bun run build
```
