= BSL NPM Library

BSL standard library for use across NPM projects.

== Installation

The project uses [https://bun.comBun] for package management and is setup with https://bun.com/docs/pm/workspaces[Bun workspaces],
simply run the following from the project root directory to install the dependencies for all
packages:

[source,bash]
----
bun install
----

== Running unit tests

To run execute the unit tests across all packages with the [Vitest](https://vitest.dev) test runner,
use the following command:

[source,bash]
----
bun run vitest
----
